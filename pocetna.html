HTML

HTML prezentacijskim ili opsinim jezikom koristim se od prilike 2 godine i u tom sam razdoblju izradio nekoliko web stranica. Uz samu izradu web stranica bavio sam se i samim dizajnom i rasporedom elemenata na web stranici, kako bi korisnicima pružio što bolje iskustvo. Neke od stranica koje sam izradio pogledajte ovdje:

  • Pavić Gradnja
  • GhostHuntAR

CSS

Od kad se bavim razvojem web stranica počeo sam koristiti css stilski jezik. Uz osnove css jezika koristim se Bootstrap frameworkom, koji mi omogućuje jednostavniju, bržu i bolju manipulacijom html elementima. Sve web stranice koje izrađujem izrađujem tako da su responzivne, odnosno da pristaju uz sve veličine zaslona, takav način rada postižem uz spominjani bootstrap.

Java Script

Java Scriptom počeo sam se baviti nedavno i uz pomoć razne literature nastojim napredovati na tom području što brže i bolje mogu kako bi svojim web stranicama omogučio razno razne dinamičke elemente.

Ruby on Rails

Moja prva Ruby on Rails aplikacija bila je ujedno i moja prva web aplikacija općenito, prvo korištenje html-a i css-a te drugih alata. Ruby on Rails koristio sam na svom završnom radu preddiplomskog studija. RoR sam koristio kao MVC framework koji mi je uvelike olakšao rad kako sam radio na dvojezičnoj stranici Pavić Gradnja

Blender

Blenderom sam se koristio za vrijeme studenstke razmjene u njemačkom gradu Bremenu. Tamo sam radio na jednom porjektu iz područja Augmented reality i u skladu s tim projektom naučio sam koristiti blender i uz pomoć blendera izrađivati 3D modele, renderirati ih, animirati i slično.
